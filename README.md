# Complaint Classification System (ML + NLP)

This project classifies customer complaints into product categories using
Machine Learning and Natural Language Processing.

## Tech Stack
- Python
- TF-IDF
- Naive Bayes
- Flask
- Postman

## Features
- Text preprocessing
- ML model training
- REST API using Flask
- Real-time prediction

## How to Run
1. Create virtual environment
2. Install requirements
3. Run app.py
4. Test API using Postman

## Dataset
Kaggle complaint dataset
